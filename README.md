
# LexiNova – AI Powered Multi-Language Translator

LexiNova is an AI-powered web application that translates text into multiple languages using modern AI models.
The application automatically detects the input language and translates it into the selected target language.

## Features

* 🌍 Multi-Language Text Translation
* 🔎 Automatic Language Detection
* 🤖 AI-Powered Translation using Groq API
* 💻 Simple and Clean Web Interface
* ⚡ Fast Response Time

## Technologies Used

* Python
* Flask
* Groq API
* Langdetect
* HTML / CSS

## Installation and Setup

1. Clone the repository

```
git clone https://github.com/yourusername/LexiNova.git
```

2. Move into the project folder

```
cd LexiNova
```

3. Install required libraries

```
pip install -r requirements.txt
```

4. Set your API key

Create a `.env` file and add:

```
GROQ_API_KEY=your_api_key_here
```

5. Run the application

```
python app.py
```

6. Open the browser and go to:

```
http://127.0.0.1:5000
```
